# Project Map: Whiteboard (CrossFit SaaS)

## Общая цель и концепт

Whiteboard — это SaaS-платформа для любителей и профессионалов кроссфита. Проект включает в себя управление тренировками, результатами, командами/организациями, профилями атлетов, а также социальные механики (лайки, комментарии, чаты).

## Технологический стек

### Frontend

- **Фреймворк:** Next.js 16 (App Router: `frontend/src/app`)
- **Библиотека:** React 19
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS
- **Стейт-менеджмент:** Zustand
- **Формы и Валидация:** react-hook-form + zod
- **PWA:** @ducanh2912/next-pwa
- **Дополнительно:** Recharts (графики), Кастомный календарь (вместо FullCalendar), DnD Kit (drag & drop), интерфейсы Radix UI, Socket.IO Client (вебсокеты).

### Backend

- **Фреймворк:** NestJS
- **Язык:** TypeScript
- **База данных:** PostgreSQL
- **ORM:** Prisma
- **Авторизация:** JWT (Access + Refresh tokens)
- **Real-time:** Socket.IO (@nestjs/websockets)
- **Фоновые задачи:** @nestjs/schedule (cron)

## Архитектура и структура

Проект разделен на два основных репозитория/папки:

1. `/frontend` — клиентская часть. Основная логика в `src/app`, переиспользуемые компоненты в `src/components`, сторы в `src/lib/store`.
2. `/backend` — серверная часть. Бизнес-логика в `src/services`, API-роуты в `src/controllers`, работа с БД в `prisma/schema.prisma`.

## Текущий функционал и прогресс

- **Пользователи и Авторизация:** Реализована ролевая модель (ATHLETE, TRAINER, ORGANIZATION_ADMIN, SUPER_ADMIN). Есть логин, регистрация с **подтверждением email** (посылка письма через SMTP), рефреш токенов, управление профилем (включая аватары, антропометрию). Структура подготовлена для OAuth (Таблица `OAuthAccount`, заглушки в AuthService). Соблюдены требования ФЗ-152 (согласия).
- **Организации и Команды:** Поддержка мульти-тенантности, создание команд, инвайт-коды для приглашения атлетов, роли внутри команд (OWNER, ADMIN, MEMBER).
- **Тренировки (Workouts/WODs):** Глобальная база тренировок (Girl, Hero WODs) и упражнений (`ContentExercise`). Планирование мероприятий (Events) со схемами (FOR_TIME, AMRAP, EMOM).
- **Дневник и Результаты:** Логирование результатов тренировок (EventResult), силовые показатели (StrengthWorkoutResult), персональные рекорды.
- **Социальные функции:** Лайки к результатам, уведомления, базовая структура для чатов.
- **Администрирование:** Админ-панель для `SUPER_ADMIN` (`/admin`), управление статусами пользователей (бан), управление контентными блоками, новости.

## Планы на развитие

- Доработка модуля чатов и коммуникации.
- Расширение виджетов на дашборде.
- Более глубокая аналитика прогресса по тренировкам.
- Поддержка новых типов тренировок и таймеров.
