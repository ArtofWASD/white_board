# Обзор функциональности проекта Whiteboard

## Основной стек
- **Frontend**: Next.js 16, React 19, Tailwind CSS, Zustand, Recharts, FullCalendar.
- **Backend**: NestJS, Prisma ORM, PostgreSQL, JWT Authentication.

## Реализованный функционал

### 1. Пользователи и Аутентификация
- **Роли**: ATHLETE, TRAINER, ORGANIZATION_ADMIN, SUPER_ADMIN.
- **Профиль**: Личные данные (рост, вес, пол), настройки дашборда.
- **Безопасность**: JWT токены, блокировка пользователей.

### 2. Организации и Команды
- **Организации**: Управление группами пользователей.
- **Команды**: Создание команд, приглашение через инвайт-коды, роли внутри команды (Owner, Admin, Member).

### 3. Тренировочный процесс (WOD & Exercises)
- **База упражнений**:
  - `UserExercise`: Пользовательские упражнения.
  - `ContentExercise`: Системная библиотека упражнений (с видео и рейтингом).
- **WOD (Workout of the Day)**:
  - Типы: GIRL, HERO, OTHER.
  - Схемы: FOR_TIME, AMRAP, EMOM.
  - Отслеживание групп мышц.
- **Результаты**:
  - Запись результатов тренировок (EventResult).
  - Трекинг силовых показателей (StrengthWorkoutResult, ExerciseRecord).

### 4. Календарь и События
- Планирование тренировок и событий.
- Статусы событий (Future, Completed, Cancelled).
- Привязка событий к командам или индивидуально.

### 5. Контент и Блог
- **Новости**: Публикация новостей с изображениями.
- **Управление контентом**: Разделение на новости, тренировки и упражнения.

### 6. Админ-панель (Super Admin)
- Полное управление пользователями и организациями (бан/разбан, смена ролей).
- Управление системными настройками (Feature Flags).
- Статистика системы.

### 7. Инструменты
- **Таймер**: Встроенный инструмент для тренировок.
- **Дашборд**: Кастомизируемые виджеты (трекинг веса, активность, упражнения).
